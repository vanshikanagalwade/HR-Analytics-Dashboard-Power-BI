HR Analytics Dashboard – Power BI Project
📌 Project Overview

This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee data and uncover insights related to attrition, performance, and workforce trends.

The goal of this dashboard is to help organizations make data-driven HR decisions by identifying key factors affecting employee retention and overall workforce efficiency.

🎯 Objectives
Analyze employee attrition trends
Identify key factors affecting employee turnover
Monitor department-wise performance
Provide actionable insights for HR decision-making
📁 Dataset Description

The dataset includes key HR-related attributes such as:
1.Employee ID
2. Age, Gender
3. Department & Job Role
4. Salary & Monthly Income
5. Years at Company
6. Job Satisfaction & Performance Rating
7. Attrition (Yes/No)

📊 Dashboard Features
🔹 Attrition Analysis (Overall & Department-wise)
🔹 Employee Demographics Visualization
🔹 Salary & Income Distribution
🔹 Job Satisfaction Insights
🔹 Experience vs Attrition Trends
🔹 Interactive Filters & Slicers
🛠️ Tools & Technologies Used
Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated measures and KPIs
📈 Key Insights
High attrition observed in specific departments and job roles
Employees with lower job satisfaction are more likely to leave
Salary and experience play a significant role in retention
Early-stage employees (0–2 years) show higher turnover
🚀 How to Use
Open the .pbix file in Power BI Desktop
Explore different report pages
Use slicers to filter data by department, gender, etc.
Analyze trends and insights interactively
💡 Conclusion

This dashboard helps HR teams to understand workforce patterns, reduce attrition, and improve employee satisfaction using data-driven insights.
